# Search movie here (Movie Store)

A cleaned Create React App structure of your Movie Explorer converted to plain CSS (dark theme).

## Quick start
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm start
   ```

## Notes
- This project uses a small in-memory mock dataset in `src/data.js` and a simulated async API in `src/api.js`.
- Styles are in `src/styles.css` (dark theme with purple accents).
